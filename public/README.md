Place your public assets here (resume, avatar, images, models, etc.).

Recommended filenames (used by the site):
- resume.pdf          -> Your resume PDF (accessible at /resume.pdf)
- avatar.jpg or avatar.png  -> Your profile image (use where needed)
- models/transformer.glb     -> 3D model for the hero scene (optional)

How to add files:
1. Copy your resume PDF to: public/resume.pdf
2. Copy your profile image to: public/avatar.jpg (or .png)
3. If using a 3D model, place it at: public/models/transformer.glb

Notes:
- Keep filenames exactly as above to ensure links work out-of-the-box.
- For best performance, optimize images and 3D models before adding them (resize, compress, gltf-pipeline, etc.).
